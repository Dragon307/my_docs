
---------------------------------------------------------------

如果下载补丁升级文件?
========================

1. 先在命令行模式下执行:

vs14-kb3165756-vs2015-update3.exe /layout

2. 程序会让你选择保存补丁更新文件(package)的路径, 等下载完成后, 再用运行的补丁文件更新 Visual Studio 2015(方法见下).

---------------------------------------------------------------

如果安装升级补丁?
=====================

See: https://stackoverflow.com/questions/38185466/visual-studio-2015-update-3-setup-blocked

---------------------------------------------------------------

一般情况下, 你直接安装补丁升级文件, 应该会看到一个错误:

KB3165756 要求存在 Visual Studio 2015 系列的成员，或遭到了计算机上的其他条件阻止。

---------------------------------------------------------------

具体的解决办法是:

1. 在命令行模式下, 先切换到上一步中下载的补丁文件的目录, 然后执行:

VS14-KB3165756.exe /uninstall  (不要使用 /repair 参数, 它是不能解决问题的)

2. 然后点击 "Repaire" / "修复" 按钮. (注意不是点 "Uninstall" / "卸载" 按钮! )

3. OK, 搞定.
