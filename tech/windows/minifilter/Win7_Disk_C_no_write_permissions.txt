win7旗舰版C盘无修改（写入）权限
===================================

1. 

C盘右键：属性 --> 安全 --> 高级 --> 所有者 --> 编辑 --> Administrators，

替换子容器勾上，

确定，然后全部确定。(会更新所有文件和文件夹的权限，时间有点长。)

注意：先关闭之前的 C 盘属性界面，一定要先关闭，才能继续下面这一步。

2.

再 C盘右键：属性 --> 安全 --> 高级 --> 权限 --> 更改权限 --> 添加，

把你的用户名打进去，勾上“完全控制”，确定；

勾上下面的“使用可从次对象继承的权限替换所有子对象权限”，确定。

(此时也会更新所有文件和文件夹的权限，时间有点长。)

Win7 Administrator 账户下，替换 C 盘文件无权限，求帮助。
===============================================================

以前改图标该过权限，不过要在管理员下执行：

1.  C盘右键 --> 属性 --> 安全 --> 高级 --> 所有者 --> 编辑 --> 选择你的帐号（点击应用，弹框点确定点完）
2.  C盘右键 --> 属性 --> 安全 --> 高级 --> 权限 --> 更改权限 --> 选择你的账户 --> 编辑 --> 完全控制 --> 确定点完
