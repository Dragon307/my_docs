
1、下载并解压：cesium-1.34：http://cesiumjs.org/releases/Cesium-1.34.zip

2、下载并安装 nodejs：https://nodejs.org/dist/v6.11.0/node-v6.11.0-x64.msi

3、使用 cmd 进入 cesium-1.34 解压主目录

4、通过 npm 安装 cesium 的必要组件:

    npm install express
    npm install compression
    npm install request
    npm install yargs

5、启动 cesium server：node server.js

6、http://localhost:8080 内含文档、示例
